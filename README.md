# calcws-ees06
Descrição
 - Uma calculadora simples implementada em Java que roda sobre um web service.

Objetivo
 - O objetivo deste projeto é a realização de duas atividades do curso de pós graduação de Engenharia de Software EAD da UFRGS, sendo que a primeira atividade é a realização de um web service simples em Java para a disciplina de SOA e a segunda atividade é a criação de um projeto em um repositório de livre escolha da disciplina de Software Livre.

Tecnologias
 - Java
 - SOA
